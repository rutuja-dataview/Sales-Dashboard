# Sales-Dashboard
This project presents an interactive Sales Performance Dashboard that analyzes business performance across sales, profit, customers, products, regions, and shipments. The dashboard provides insights into yearly trends, customer segments, product categories, regional sales, shipping costs, delivery timelines, and seasonal performance,enabling data-driven decision-making and business optimization.
